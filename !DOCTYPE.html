Discernment of Liver Disease Analysis
with Clinical Data Using Differential Evolution

This system predicts liver disease in patients using a hybrid machine learning approach combining Differential Evolution (DE) for feature selection and a Back Propagation Neural Network (BPNN) for classification.


The Non-Alcoholic Fatty Liver Dataset (113 patient records, 12 clinical features) is used. DE selects the most diagnostically significant features — ALT (SGPT) and Albumin — achieving a best fitness of 0.9967. The BPNN then classifies liver disease status with 96.55% Accuracy and 100% Recall.


Please Register and Login to access the prediction modules.

🧬 Differential Evolution

Population-based optimization algorithm that selects optimal feature subsets. Pop=30, F=0.8, CR=0.9, Generations=50. Best Fitness = 0.9967.

🧠 Back Propagation Neural Network

Multi-layer feedforward network (2→12→8→1) trained on DE-selected features using gradient descent with momentum. R² = 0.9336.

📊 Dataset

Non-Alcoholic Fatty Liver Dataset. 113 patient records, 12 clinical features including ALT, AST, GGT, Bilirubin, Albumin, BMI, Age.

🎯 Performance

Conclusion
The proposed DE + BPNN hybrid system achieves superior performance compared to all existing algorithms. Differential Evolution reduces 12 features to 2 optimal features (ALT and Albumin) with fitness 0.9967. The BPNN achieves 96.55% accuracy and 100% recall, ensuring zero missed liver disease diagnoses. The proposed system outperforms Random Forest by +14%, and Decision Tree/XGBoost by +5.25%.
